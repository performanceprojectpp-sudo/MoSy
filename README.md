# MoSy — Performance Project PWA

## Come installare su Android (Xiaomi e tutti gli altri)

### Metodo 1 — GitHub Pages (GRATUITO, consigliato)

1. Vai su https://github.com e crea un account gratuito
2. Clicca **"New repository"**
3. Nome: `MoSy` — metti spunta su **Public** — clicca **Create**
4. Clicca **"uploading an existing file"**
5. Trascina TUTTI i file di questa cartella:
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
   - logo1.jpg
   - logo2.jpg
6. Clicca **"Commit changes"**
7. Vai in **Settings → Pages → Branch: main → Save**
8. Dopo 1 minuto l'app sarà su: `https://TUOUSERNAME.github.io/MoSy`

### Installazione su Xiaomi/Android
1. Apri Chrome sul telefono
2. Vai all'URL sopra
3. Chrome mostrerà in basso: **"Aggiungi MoSy alla schermata Home"**
4. Tocca **Installa** → l'icona appare come un'app vera!

### Funziona OFFLINE
Dopo la prima apertura, l'app funziona anche senza internet.
I dati vengono salvati localmente sul telefono.
